# zure

Zig binding for rust **regex** engine.

### Usage

Build rust regex library:
```sh
$ git clone git://github.com/rust-lang/regex
$ cd regex/regex-capi
$ cargo build --release # it exist in ../target/release [librure.so, librure.a]
```
